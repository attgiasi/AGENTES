# Guia de uso para leigo

## O jeito mais seguro de começar

1. Abra o painel.
2. Deixe `Simulação / não alterar Gmail` ligado.
3. Use autonomia nível 2.
4. Clique em `Rodar agora`.
5. Veja logs e relatórios.

## Para começar a organizar

1. Use autonomia nível 3.
2. Deixe etiquetas ligadas.
3. Deixe arquivamento desligado no primeiro teste.

## Para arquivar newsletters

1. Desligue simulação.
2. Ligue `Arquivamento automático`.
3. Ligue `Arquivar emails`.
4. Use autonomia nível 4 se quiser aprovar antes.
5. Use autonomia nível 5 se quiser arquivar automaticamente.

## Se algo parecer errado

Clique em `Parada de emergência`.

Depois revise logs, permissões e remetentes protegidos.

## GitHub

O painel local salva configuração no seu computador.

O GitHub Actions usa o Secret:

```text
AGENT_SETTINGS_JSON
```

Se quiser mudar o comportamento no GitHub, atualize esse Secret.
